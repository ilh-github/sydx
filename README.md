# 插件使用说明：

## 该项目主要用于提供 中石油远程教育（北京校区）脚本服务。



## 项目分为两大类

-  浏览器-网页-在线网课插件（用于浏览器的在线作业，视频）和自动化平台二选其一：群文件 
-  桌面端-软件-在线网课自动化（用于浏览器的在线作业，视频）的自动化，和浏览器端的插件二选其一：群文件 
-  桌面端-软件-网络视频考试（用于人像考试）：    群文件
